===任务池系统===

线上正式部署说明：

任务池系统当前部署在 10.0.0.16 虚拟机上，该虚拟机位于 10.0.0.8 实体机上，虚拟机名称 riverlet-01。
任务池原则上每个部门一套程序，但程序的共用核心程序部署在 /usr/local/taskpool 中，各部门程序通过自己的软连接引用此目录。
各部门的目录（arch\tech_0\tech_1....）中只存储本部门特定的配置文件。
各部门的目录均位于 /home 下。


线上测试部署说明：
测试目录位于 /home/taskpool ，该测试环境中的所有文件均独立，不与其他部门共用核心程序。

数据库：
此工程测试、正式数据库均为 10.0.0.16 本机的MySQL。
数据会每天一次自动备份到 10.0.0.8 中，通过10.0.0.8上的crontab执行 /data/backup 中的脚本实现。


任务池定时分发程序运行方法（以arch部门的为例）：
/usr/local/php-5.4.4/bin/php /home/arch/public_html/yiic timer start &
所有命令已添加到 /etc/rc.local 文件中，默认情况下系统启动后即可自动运行任务池系统，无需人工干预。
